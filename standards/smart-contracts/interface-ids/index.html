Interfaces Ids

Interfaces Ids helps to check if a contract supports a specific interface, it's a meta-interface so to say. This is useful if we want to interact with a contract but we don't know if it supports an interface such as ERC725Y or LSP1UniversalReceiver, etc.

The supportsInterface function from ERC165 Standard returns TRUE if the standard is supported, FALSE otherwise.

note

Interface Ids are not the best secure way to ensure that a contract implements specific functions as its manually set and could be set by anyone to any value.
